Convert Numbers File to CSV

Quick Conversion Steps:

Step 1: Open your Numbers file on Mac/iPad
Step 2: Click "File" → "Export To" → "CSV..."
Step 3: Choose "UTF-8" encoding and click "Next"
Step 4: Save the file and upload the CSV version here
Alternative: You can also copy the data from Numbers and paste it into a text editor, then save as .csv file
Note: Make sure to fix the duplicate equipment code PMP-CEN-006 before uploading. The "Shaft Misalignment" entry should have a unique code like PMP-CEN-099.
